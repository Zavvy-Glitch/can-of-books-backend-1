# Can of Books

**Authors**: Justin Hamerly, Tray Chea, Kason Braley

**Version**: 1.0.0 

## Overview
An application to track what books have impacted you, and what’s recommended to read next.

## Getting Started
1. Clone the project
2. `cd` into the local project folder
3. run `npm install`
4. run `npm start`
